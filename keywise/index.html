Frequently Asked Questions

Does KeyWise sync to cloud services?

No. Current production builds are on-device and do not provide cloud sync or Google Drive sync features.

Can I create custom credential types and custom fields?

Yes. You can create custom credential types and define custom fields so each record layout fits your own account data model.

Is Premium a subscription?

No. Premium is a one-time purchase that raises vault and credential limits. There are no monthly or yearly renewals.

Can I use KeyWise fully offline?

Yes. Core vault flows are designed for offline use. Your encrypted data remains on-device unless you explicitly export backup files.

How do backup and restore work?

KeyWise lets you create encrypted backup files that you keep yourself. Restore is manual and intentional, so you choose where your backup file is stored and when it is used.

What happens if I change phones?

If you migrate without a backup, on-device data will not follow automatically. Export an encrypted backup first, then import it on the new device.
